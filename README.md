**Overview
**
This project is a simple React dashboard where users can upload a diagram image and view detected circuit components in a sidebar.

The application demonstrates React fundamentals including component-based architecture, state management using hooks, and responsive UI design.

**Installation
**
**Clone the repository:
**
git clone https://github.com/YOUR_USERNAME/diagram-dashboard.git

**Navigate to the project folder:
**
cd diagram-dashboard

**Install dependencies:
**
npm install

**Run the project:
**
npm start

**The app will run on:
**
http://localhost:3000
